Create a CDI bean `OutbreakDetector` with a method `detect(String county, String disease)` that:

- Queries the last 4 weeks of DiseaseReport for that county and disease via DiseaseReportRepository (write a custom query method: findLast4WeeksByCountyAndDisease).
- Groups by calendar week, sums cases per week.
- Computes mean and standard deviation of the 4 weeks.
- Gets the latest week's cases.
- If latest > mean + 2*stddev, returns an `OutbreakAlert` object (not yet saved) with:
  riskScore = min(100, ((latest - mean) / (2*stddev)) * 100)
  alertLevel = based on zScore: 1-2=YELLOW, 2-3=ORANGE, >3=RED
  zScore = (latest - mean) / stddev
- Otherwise returns null.
