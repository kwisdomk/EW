Generate OpenShift resources (single YAML) for:

- DeploymentConfig for backend, using the Dockerfile, with env placeholders for DATABASE_URL, WATSON_API_KEY, WATSON_URL.
- Service for backend (port 8080).
- Route for backend (public).
- DeploymentConfig for frontend (nginx serving built static files from /frontend/dist).
- Service and Route for frontend.
- PostgreSQL ephemeral DeploymentConfig and Service.
